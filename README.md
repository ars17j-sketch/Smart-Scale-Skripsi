# Smart Scale Skripsi

Proyek timbangan dan pengukur tinggi badan IoT berbasis ESP32.

## Hardware
- ESP32 DevKit V1
- HX711_ADC + 4x Load Cell 50 kg
- VL53L0X
- LCD 20x4 I2C
- Buzzer
- EEPROM
- Blynk IoT

## Fitur
- Pengukuran berat badan
- Pengukuran tinggi badan
- Perhitungan BMI
- Multi User
- Auto Tare
- Auto Kalibrasi
- Median Filter
- Moving Average
